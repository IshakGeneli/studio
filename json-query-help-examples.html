Sorgu Konsolu Örnekleri

JavaScript syntax'ı kullanarak verilerinizi filtreleyebilir ve dönüştürebilirsiniz.

1. Basit Filtreleme

ID'si 5'ten büyük olan öğeleri listeler.

items.filter(x => x.id > 5)

2. Veri Dönüştürme (Mapping)

Sadece ürün isimlerinden oluşan yeni bir liste oluşturur.

products.map(p => p.name)

3. Koşullu Arama

Kategorisi 'elektronik' olan ilk öğeyi bulur.

store.inventory.find(item => item.category === 'elektronik')

Daha fazla özellik için JavaScript Array metotlarını (reduce, sort, slice vb.) inceleyebilirsiniz.
